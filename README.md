#####################        ENGLISH

                                    HOW TO INSTALL:
    $ chmod +x install.sh
    $ ./install.sh
      Done
    $ compile -h
    
All done. So, you have to run the 'install.sh' script.
After the last command you should see the help message. Use those to get info!

To change language use this command:

    $ compile --set --select-language
